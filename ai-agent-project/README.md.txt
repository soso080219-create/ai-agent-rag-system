# AI Agent RAG System

## 📌 Overview
LLM-based RAG pipeline for real-time situation understanding and decision-making in a robot Q&A simulation system.

## 🛠 Tech Stack
- Python
- FastAPI
- LangGraph
- Ollama
- LLM (Claude / etc.)

## 🚀 Features
- Real-time question answering based on contextual data
- RAG pipeline with embedding and retrieval
- Multi-step reasoning using agent flow
- Chunking-based document search optimization

## 💡 What I Did
- Designed and implemented FastAPI backend server
- Built RAG pipeline with embedding and retrieval logic
- Developed agent workflow using LangGraph
- Optimized response quality with chunking strategy

## 📈 Results
- Improved response accuracy and reliability
- Implemented real-time decision-making flow
- Enhanced user experience through structured responses
